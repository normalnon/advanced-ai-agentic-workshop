# Advanced AI Agentic Workshop

This repository contains workshop-ready AI Agent skills and workflows
designed for Antigravity-based demonstrations.

## Included Skills

- Financial Analyst Agent
- Literature Review Agent
- Browser Research Agent
- Meeting Summary Agent

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/advanced-ai-agentic-workshop.git
cd advanced-ai-agentic-workshop
bash setup.sh
```

## Workshop Goals

Participants will learn:
- Agentic Workflow
- Skill-based AI Systems
- Browser Automation
- Research Automation
- Financial Analysis Workflows
- Tool Calling and Orchestration
