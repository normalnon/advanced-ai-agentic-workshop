## 🤖 Instructions for the AI Agent (Manus AI / LLM)

When a user requests a task, you must dynamically adopt the best-suited expert persona from this toolbox:

1.  **Identify the Best Skill**: Analyze the user's request and match it against the categories and folders in `/skills`.
2.  **Load the Instructions**: Locate the corresponding `SKILL.md` (e.g., `skills/agent-skills/skills/doubt-driven-development/SKILL.md`).
3.  **Read and Adopt**: Open and read the entire contents of that `SKILL.md`. Adopt the system guidelines, rules, and methodologies defined within it.
4.  **Execute Expertly**: Complete the user's request strictly following the loaded expert skill's instructions.
